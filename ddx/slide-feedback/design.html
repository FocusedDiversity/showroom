Visual

  • Small "Feedback" pill button added to the topbar, right side, before the Synaptiq logo
  • Chat bubble icon + "Feedback" label — unobtrusive, doesn't compete with slide content
  • When collapsed, the iframe takes the full width of the viewport — no panel visible

Interaction

  • Clicking the Feedback button opens the right-side feedback panel (see next screen)
  • Button is always visible regardless of which slide is active

2. Deck Viewer — Feedback Panel (Open)

Deck Viewer — Feedback Open
AI Solutions Overview 3 / 12
SYNAPTIQ
▶
Slide deck content
Feedback for Slide 3

No feedback on this slide yet.

Visual

  • Feedback panel is a 320px-wide right sidebar, separated from the iframe by a left border
  • Panel has three zones: header (slide label + close), scrollable middle (prior comments), and input pinned to bottom
  • Iframe shrinks horizontally to make room — the slide content is never covered or obscured
  • "Feedback for Slide N" label updates dynamically as the viewer navigates slides
  • Topbar Feedback button switches to active state (blue highlight) when panel is open

Interaction

  • Slide number in the label updates automatically when the viewer navigates to a different slide
  • Pressing Enter or clicking Send submits the feedback
  • Clicking X or the Feedback button again closes the panel and iframe returns to full width
  • Input is focused automatically when the panel opens

Visual

  • Prior comments listed in the scrollable middle area of the right panel
  • Each comment in a light gray card with timestamp
  • More vertical room for comments than a bottom panel — can show several without scrolling
  • Input placeholder changes to "Add another comment..."

Interaction

  • Viewer can submit multiple comments per slide — each is stored separately
  • Prior comments are read-only (no editing or deleting in v1)
  • Panel scrolls vertically if there are many prior comments — input stays pinned at bottom
